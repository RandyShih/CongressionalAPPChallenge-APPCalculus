Short Summary
Now, as you have seen from the previous article, we can simplify single fractions by cancelling out factors, but what if we have multiple fractions? For example: $$\color{#04afff}{\frac{1}{x^2+4x+3} = \frac{1}{(x+1)} + \frac{2}{(x+3)}}$$ Well, we can begin simplifying this equation by factoring the denominator of the first fraction, which would be: $$\color{#04afff}{x^2+4x+3\\\\x^2 + 3x + 1x + 3\\\\x(x+3)1(x+3)\\\\(x+1)(x+3)\\\\\frac{1}{(x+1)(x+3)}}$$ Now that the first fraction is simplified, we can simply it and the other fractions even more now by multiplying both sides by the denominator of the first fraction $$\color{#04afff}{\frac{(x+1)(x+3)}{(x+1)(x+3)}=\frac{(x+1)(x+3)}{x+1} + \frac{2(x+1)(x+3)}{x+3}}$$ Now just simplify and cancel out the factors that you can for every fraction. $$\color{#04afff}{1 = (x+3) + 2(x+1) \\\\ 1 = x + 3 + 2x + 2 \\\\ -3x - 4 = 0}$$ This is the simplified value of the entire equation, which is useful for easily finding the zeros of the function through solving for x.
