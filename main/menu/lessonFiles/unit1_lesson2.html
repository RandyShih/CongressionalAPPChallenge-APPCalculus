Short Summary
The derivation of a function is essentially the slope of a line at a certain point, and you find the derivative or slope of a function at any point, you must first find the derivative of the initial function. For basic derivation of exponential functions, multiply the function by the value of the exponent, and then subtract the value of the exponent by 1. $${\color{#04afff}f(x) = bx^{c} \\ \color{#04afff}c, b \in \mathbb{R} \\ \color{#04afff}\frac{dx}{dy} = c * bx^{c - 1}}$$ For instance, if you choose a random point in a graph, such as $${\color{#04afff}f(x) = 2x^{3}}$$ then the derivative at any point o graph will be $${\color{#04afff}\frac{dx}{dy} = 6x^{2}}$$ This results in said random point's derivative being the value of the derivation function with the point plugged into it.
Finding the derivative of a function with this method is only applicable to the first function presented in this summary, which could also be a polynomial function as long as each term abides by the format of the function. We will explore how to derive other functions using other methods in the next couple of videos.
