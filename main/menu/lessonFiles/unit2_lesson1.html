Short Summary
To begin with, the zeros of a polynomial are essentially where the function intercepts the x-axis, whose usefulness will be apparent in solving questions pertaining to when the value of the function reaches 0. To find the zeros of a polynomial, we have to first factor it. So let us assume the polynomial: $$\color{#04afff}{x^2+5x+4}$$ First, factor out this polynomial, which would be: $$\color{#04afff}{x^2+4x+1x+4 \\\\ x(x+4)1(x+4) \\\\ (x+1)(x+4)}$$ For each factor of the factored polynomial, set them to zero and solve for x to find the zeros of the polynomial. $$\color{#04afff}{x + 4 = 0,\, x = -4 \\\\ x + 1 = 0 ,\, x=-1}$$ This means that the zeros of the polynomial in this example is both x = -1 and x = -4, which you can see if you graph it out.
